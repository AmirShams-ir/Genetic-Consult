# سامانه مشاوره ژنتیک

![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)
![Platform](https://img.shields.io/badge/Platform-Windows-blue)
![Language](https://img.shields.io/badge/VB.NET-.NET_Framework-purple)

سامانه مشاوره ژنتیک یک نرم‌افزار تخصصی جهت مدیریت پرونده بیماران، ثبت اطلاعات مشاوره ژنتیک، رسم شجره‌نامه (Pedigree) و ارائه خدمات مشاوره ژنتیک می‌باشد.

این نرم‌افزار در ابتدا به عنوان بخشی از یک پروژه پژوهشی دانشگاهی در **دانشگاه شهرکرد** طراحی و توسعه یافته است.

---

# امکانات

- مدیریت پرونده بیماران
- ثبت اطلاعات مشاوره ژنتیک
- رسم شجره‌نامه خانوادگی
- مدیریت تصاویر و مدارک پزشکی
- تهیه گزارش‌های پزشکی
- پایگاه داده محلی
- سیستم ورود کاربران
- رابط کاربری فارسی

---

# فناوری‌های استفاده شده

- Visual Basic .NET
- .NET Framework
- Microsoft Access Database
- Windows Forms

---

# وضعیت پروژه

نسخه موجود، نسخه کلاسیک مبتنی بر VB.NET می‌باشد.

در آینده نسخه جدید این سامانه با فناوری‌های مدرن توسعه خواهد یافت، از جمله:

- Python
- FastAPI
- PostgreSQL
- Vue.js
- هوش مصنوعی
- رابط کاربری چندسکویی
